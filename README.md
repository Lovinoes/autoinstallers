# Autoinstallers
A collection of shell-based installation scripts to deploy game servers using Docker Compose.

Each installer is designed to provide a fast, repeatable setup with sane defaults, minimal user input, and container-based deployment.

## Available Installers

### [Minecraft Server (Click for more Info)](/minecraft)
Automated installer for a PaperMC Minecraft Java Edition server using Docker.

```bash
curl -fsSL https://raw.githubusercontent.com/Lovinoes/autoinstallers/main/minecraft/scripts/installer.sh -o installer.sh
sudo bash installer.sh
```
